In Teamcenter set the follow properties (if it doesn't exist, create it!):
ActiveWorkspaceHosting.NX.URL, it must have a value equal to the AWC network address, for example: http://awc_srv01:3000

Also checking and adjusting (related with official documentation) the followings properties:
AWC_NX_AddComponentSupportedTypes
AWC_NX_ApplicationAndEnvironmentIsSupported
AWC_NX_OpenRelationBrowserSupportedTypes
AWC_NX_OpenRequirementLinksSupportedTypes
AWC_NX_OpenSupportedTypes
AWC_NX_SupportsContent
